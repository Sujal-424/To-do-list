This is a simple to do list web app.
The basic concept of this app is to provide
the user with a simple to do list where they 
can add their own to do's with the time required
and priority of the task.
After every 3 hours the user will be notified about
all the tasks that are still to be completed and
all the tasks that are not completed.
It will also show the user statistics as well.
